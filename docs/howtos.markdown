## Create FAQs

![](./FAQErstellen.png)

1. Create FAQ.
2. Give Title (the real question).
3. Check if the classic Editor is active
4. The answer to the question
5. the order of the FAQs
6. intertactive Button to linked site
7. FAQ-Category
8. Save or direct publishing of the FAQ
